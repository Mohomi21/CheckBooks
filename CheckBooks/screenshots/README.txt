This folder is for UI screenshots of the app.

Place your screenshots here and update README.md image links accordingly.

Suggested filenames:
- main_ui.png          -> screenshot of the MainActivity UI (layout before running)
- main_running.png     -> screenshot of MainActivity while the app is running and showing a toast
- screen2_ui.png       -> screenshot of Screen2 UI (layout before running)
- screen2_running.png  -> screenshot of Screen2 while the app is running (RecyclerView visible)
